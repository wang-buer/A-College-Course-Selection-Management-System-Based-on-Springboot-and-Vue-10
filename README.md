# 基于springboot和vue的高校选课管理系统10



## 1、项目介绍

基于springboot和vue的高校选课管理系统10拥有两种角色

教师：发布课程、成绩登记、课程考勤、课程表等

学生：选课退课、登录注册、找回密码、查看成绩和课程表等

## 2、项目技术

后端框架：springboot、mybatisplus

前端技术：VUE

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
